# Calcuator - JavaSscript

This program is a simple calculator that was used to deploy an application to a web service (Azure) for a university module (Cloud Computing Development) at Leeds Beckett University.

## How to Use 

It's simple to get this program running on any device running a web browser. 

### What you'll need

```
A Web Browser (Safari, Chrome, FireFox)
```

```
A Text Editor (Atom, Visual Studio Code)
```

### How to Run

```
Download the ZIP for this program
```

```
Open the ZIP Directory in your Text Editor 
```

```
Edit or make changes via index.html
```

```
Run the calculator via opening index.html in a web browser.
```

## Deployment

This will work on any live system running a web browser.
